Online Banking Application

I've created a program for Banking System in C++ You are given 9 options: Open Account Close Account Show All Accounts Deposit in Account Withdraw from Account Transfer to an Account Show Balance Show All Transactions Quit My program stores the deposits, and withdrawals, and transfers in a a file and uses them when the program is run again. Note: The currency is in Indian rupees.
